APP2 - Aplicação web de compra de ofertas

Aplicação WEB como Peixe Urbano utilizando JavaScript ES6, TypeScript e Angular! Neste projeto foram criados serviços, promises, observables, pipes e formulário reativo! Pesquisa de ofertas com consumo de API Rest (JSON Server) e rotas de navegação. Pacotes instalados: Angular-CLI, Ng-Bootstrap (Angular), Font Awesome e Animate.
