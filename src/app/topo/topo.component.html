#APP2 - Aplicação WEB como Peixe Urbano!

Curso de Desenvolvimento WEB com ES6, TypeScript e Angular

Este projeto é baseado no famoso site Peixe Urbano, onde pesquisa e simula a compra de ofertas via formulário reativo com o API REST fake (JSON Server).

Processos de desenvolvimento abordados:
  • Como criar e consumir serviços;
  • Promises;
  • Como consumir uma API Rest;
  • Rotas e navegação;
  • Observables;
  • Programação Reativa;
  • Pipes;
  • Reactive forms;
  • Font Awesome e Animate.css
